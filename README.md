# Markdown to HTML compiler

![](https://github.com/mikeizbicki/markdown-compiler/workflows/doctests/badge.svg)
![](https://github.com/mikeizbicki/markdown-compiler/workflows/flake8/badge.svg)
![](https://github.com/mikeizbicki/markdown-compiler/workflows/command_line/badge.svg)

A simple project for converting markdown files to HTML.

Basica usage:
```
$ markdown-compiler example/README.md
```

Fancy CSS formatting can be included with the flag `--add_css`:
```
$ markdown-compiler example/README.md --add_css
```
